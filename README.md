# プログラミング技術（Python入門）

## 学習内容

### Raspberry Pi Picoを使った組み込みプログラミング

Raspberry Pi Picoを使って、Python3と互換性がある言語MicroPythonで、マイクロコンピュータでのLEDの制御や液晶ディスプレイ`[SSD1306]`と温湿度・気圧センサ`[AE-BME280]`の制御などの、組み込み向けのPythonプログラミングを学習します。本学習は、text.mdを参照してください。
